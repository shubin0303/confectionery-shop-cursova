# Sweet Delights - Кондитерский магазин

Статический сайт кондитерского магазина с функционалом:
- Информация о магазине
- Каталог продукции (12+ наименований)
- Система фильтрации товаров
- Информация о наличии и ценах
- Страница заказа и доставки
- Интеграция с "базой данных"

## Файлы проекта

Все файлы находятся на одном уровне:
- `index.html` - Главная страница
- `about.html` - О магазине (справка, контакты, карта)
- `products.html` - Каталог продукции
- `order.html` - Заказ и доставка
- `database.js` - База данных продуктов
- `styles.css` - Основные стили
- `script.js` - Общие скрипты
- `products-script.js` - Скрипты для страницы продуктов
- `README.md` - Эта инструкция

## Как запустить на GitHub Pages

### Способ 1: Через настройки репозитория

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы проекта в корень репозитория
3. Перейдите в Settings → Pages
4. В разделе "Source" выберите:
   - Branch: `main`
   - Folder: `/ (root)`
5. Нажмите "Save"
6. Через несколько минут сайт будет доступен по адресу:
   `https://ваш-username.github.io/название-репозитория/`

```bash
# Клонируйте репозиторий
